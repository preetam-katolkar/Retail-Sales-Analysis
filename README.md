# Retail Sales Analysis

## Project Overview
This project is an end-to-end Retail Sales Analytics Dashboard built using Power BI.
It provides actionable insights into overall sales performance, category and segment contribution, and profitability analysis.

## Dataset
The dataset includes order-level, customer, product, sales, profit, and region information.
It has been cleaned and prepared for analysis, including date tables, calculated measures, and data modeling.

## Dashboard Pages
1. **Sales Performance Overview** – Total sales, profit, orders, and trends over time
2. **Sales Contribution Analysis** – Category and segment-wise sales performance
3. **Profitability & Loss Analysis** – Identification of loss-making regions and sub-categories

## Key Insights
- Overall sales and profit trends over time
- Top-performing categories and segments
- Loss-making regions and sub-categories highlighted
- Interactive slicers for date and region analysis

## Tools Used
- Power BI Desktop
- DAX (Measures & Calculations)
